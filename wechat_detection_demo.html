💰 微信浏览器检测功能演示

智能检测、用户友好、记忆功能、动画效果、响应式设计

🧪 功能测试

点击下面的按钮来测试不同的功能：

🎯 智能检测

只在微信浏览器中显示提示，不会在普通浏览器或小程序环境中干扰用户。

// 检测微信浏览器 function isWechatBrowser() { var ua = navigator.userAgent.toLowerCase(); return ua.indexOf('micromessenger') !== -1; }

👤 用户友好

用户可以手动关闭提示，右上角有关闭按钮，点击即可关闭。

// 关闭按钮 <button class="wechat-close" onclick="closeWechatWarning()" title="关闭提示">×</button>

💾 记忆功能

使用localStorage记住用户的选择，关闭后不再显示提示。

// 检查是否已关闭 function isWarningDismissed() { return localStorage.getItem('wechat_warning_dismissed') === 'true'; }

✨ 动画效果

平滑的滑入动画和脉动提醒，提升用户体验。

// CSS动画 @keyframes slideInDown { from { transform: translateY(-100%); opacity: 0; } to { transform: translateY(0); opacity: 1; } }

📱 响应式设计

适配手机屏幕，在不同设备上都有良好的显示效果。

// 响应式媒体查询 @media (max-width: 768px) { .wechat-warning { margin: 0.5rem; } }

🛡️ 小程序安全

检测小程序环境，避免在小程序中显示不必要的提示。

// 检测小程序环境 function isMiniprogram() { return ua.indexOf('miniprogram') !== -1 || window.__wxjs_environment === 'miniprogram'; }

📋 使用说明

  1. 自动检测：应用会自动检测用户是否在微信浏览器中访问
  2. 智能显示：只在微信浏览器中显示提示，其他环境不干扰
  3. 用户控制：用户可以点击右上角×按钮关闭提示
  4. 记忆功能：关闭后使用localStorage记住，不再重复显示
  5. 动画效果：平滑的滑入动画和脉动效果吸引注意
  6. 响应式：适配各种屏幕尺寸，手机端优化显示
